"...프로그래밍 학습이라고 하는 것이 누군가한테 가르침을 받아서 그 가르침만으로 어느 수준까지 올라간다는 건 사실상 어렵다." → 스스로 자가 학습하는 역량이 필요한 이유. (+습관화)
