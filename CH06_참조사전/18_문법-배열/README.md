이름이 있는 데이터 유형과
이름이 없고 위치 값을 갖고 있는 데이터 유형.

**slice**

원형은 복사해서 그대로 두고, 오린(!) 값만 가져온다.(반환한다)
두 번째 인자를 포함하지 않고 빼온다.

**splice**

원형도 변형시킨다.
두 번째 인자를 포함해서 빼온다.

헷갈려 😅
